分辨率与清晰度

近期闲聊，发现对分辨率的概念不是很清晰，于是做了进一步进行了了解

分辨率

解析度（英语：resolution）又称分辨率（resolving power），泛指量测或显示系统对细节的分辨能力。此概念可以用时间、空间等领域的量测。影像分辨率（image resolution）则专指影像、图像记录物体细部能力的一种度量，此时又称解像度、解像力。分辨率是通过将图像高度的像素数乘以宽度的像素数来决定的。1

日常用语中之分辨率，多指影像“清晰度”（definition）。分辨率越高代表影像质量越好，越能表现出更多的细节；但相对的，因为纪录的信息越多，文件也就会越大。个人电脑里的影像，可以使用影像处理软件（例如Adobe Photoshop、PhotoImpact、Gimp）调整大小、提高影片分辨率、编辑照片等。

像素 Pixel

像素或画素，为影像显示的基本单位，译自英文“pixel”，pix是英语单词 picture 的常用简写，加上英语单词“元素”element，就得到pixel，故“像素”表示“画像元素”之意，有时亦被称为pel（picture element）。每个这样的消息元素不是一个点或者一个方块，而是一个抽象的取样。仔细处理的话，一幅影像中的像素可以在任何尺度上看起来都不像分离的点或者方块；但是在很多情况下，它们采用点或者方块显示。每个像素可有各自的颜色值，可采三原色显示，因而又分成红色、绿色、蓝色三种子像素（RGB色域），或者青色、洋红色（紫色）、黄色和黑色（CMYK色域，印刷行业以及打印机中常见）。照片是一个个取样点的集合，在影像没有经过不正确的/有损的压缩或相机镜头合适的前提下，单位面积内的像素越多代表分辨率越高，所显示的影像就会接近于真实物体。

像素尺寸 (Pixel Dimensions)：这是最常见的方式，格式为“宽 x 高”。例如，一张分辨率为 1920 x 1080 的图片，意味着它在水平方向上有 1920 个像素，在垂直方向上有 1080 个像素。这个数字越高，意味着图像包含的像素总数越多，细节也就越丰富。

像素密度 (Pixel Density)：这个指标描述的是在单位物理尺寸内包含了多少像素，常用单位是 PPI (Pixels Per Inch)，即每英寸像素数。在屏幕尺寸相同的情况下，PPI 越高，意味着像素点越小、越密集，画面自然也就越细腻、越清晰。例如，两款同样是 6 英寸屏幕的手机，一款分辨率为 1080p，另一款为 1440p，后者的 PPI 更高，显示效果会更锐利。

PPI 计算公式

PPI=(w2+h2)dPPI=\frac{\sqrt{(w^2+h^2)}}{d}PPI=d(w2+h2)​​
  • 分子利用勾股定理计算出屏幕对角线上总共有多少个像素
  • 分母表示屏幕的物理对角线长度，单位为英寸
  • 如此一来便可知道，单位英寸上有多少个像素

例：以 2560 x 1440 的分辨率的 27 英寸的显示器为例，其 PPI 为：108

屏幕清晰吗？

从上面的分析，可以知道，像素密度是更好的衡量屏幕细腻程度的方式。另外观看距离也决定了我们需要的像素密度，即 PPI 的级别

  • 手机： PPI 高达 460+。因为我们把它举到离眼睛 20-30 厘米的地方看，需要极高的像素密度才能“欺骗”我们的眼睛，让它看不到像素点。
  • 电视： PPI 可能只有 100-150。因为我们坐在 2-3 米外看，我们的眼睛分辨能力有限，根本无法看清那么远处的单个像素。即使 PPI 不高，在正常观看距离下，画面也已经足够清晰细腻了。
